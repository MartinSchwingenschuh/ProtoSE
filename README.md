# ProtoSE
ProtoSE is a cloud storage solution based on the [Sophos](https://gitlab.com/sse/sophos "Sophos") cryptography scheme. Where you can securely store and search PDF files on a server.
Sophos is a SSE(symmetric searchable cryptohraphy) construction used as the underlying cryptographic primitive.

## Disclamer
This is a research project and as of yet not proven to be stable or secure.
